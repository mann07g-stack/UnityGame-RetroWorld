# UnityGame-RetroWorld

RetroWorld is a Unity game project built around a collection of small gameplay experiences, prototype mechanics, and stylized assets. The project is actively being developed and is not complete yet.

## Status

This project is still building. Features, scenes, balancing, and polish are in progress, and some systems may change as development continues.

## What is in the project

- Unity project structure and core game assets
- Multiple mini-game prototypes and supporting scripts
- Input, scene, audio, and UI resources in progress

## Development note

This repository is meant to track the game as it evolves. Expect unfinished content, placeholder systems, and assets that may be replaced or refined later.
